PROJECT: Plan-AI

Service Definition

사용자의 여행 취향(기간, 테마, 인원 등)을 분석하여, Gemini AI가 최적화된 여행 계획표를 제안하는 웹 서비스입니다.
AI가 생성한 데이터를 정제하여, 보기 편한 카드 뉴스 형태의 스케줄 UI로 제공합니다.

Tech Stack

React FastAPI Gemini 2.5 Flash Vercel

Problem: Unstructured Data

생성형 AI(LLM)는 기본적으로 비정형 텍스트(줄글)를 반환합니다.
하지만 실제 서비스 UI에 적용하기 위해서는 엄격하게 구조화된 JSON 데이터가 필수적이었습니다.

"AI의 답변 형식이 매번 달라져, 앱이 멈추거나 UI가 깨지는 현상 발생."

Solution: Data Engineering

프롬프트 엔지니어링과 데이터 파싱 로직을 결합하여 해결했습니다.

1. System Prompt: AI에게 JSON 스키마를 강제하여 정해진 포맷으로만 답변하도록 유도.
2. Validation: 백엔드(FastAPI)에서 데이터 무결성을 2차 검증하여 UI 렌더링 오류를 원천 차단.

Lessons Learned

"API를 연결하는 것보다 중요한 것은 '데이터를 다루는 능력'이었습니다.
AI의 불확실한 결과를 정형화된 데이터로 변환하여 서비스 안정성을 확보하는 과정에서,
프롬프트 엔지니어링이 단순한 질문이 아니라 일종의 '코딩'임을 깨달았습니다."
